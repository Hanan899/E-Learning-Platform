# E-Learning Platform

Monorepo for a school-focused E-Learning Platform built with React, Express, PostgreSQL, Sequelize, JWT, Tailwind CSS, and Multer.

## Structure

```text
elearning/
├── backend/
└── frontend/
```

## Quick Start

1. Backend:
   - `cd backend`
   - `npm install`
   - Update `.env` with your PostgreSQL credentials
   - `npm run db:sync`
   - `npm run seed`
   - `npm run dev`
2. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

## Test Commands

- Backend: `cd backend && npm test`
- Frontend: `cd frontend && npm test`
